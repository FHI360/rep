Description
-----------
This module adds a Website file type to Media module.

Requirements
------------
Drupal 7.x

Installation
------------
1. Copy the entire media_website directory the Drupal sites/all/modules/custom directory.

2. Login as an administrator. Enable the module in the "Administer" -> "Modules"
